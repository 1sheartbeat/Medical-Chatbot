About MediAssist

AI-Powered Medical Assistant | Llama 3 8B | Task-Oriented Medical Chatbot

What is MediAssist?

MediAssist is a task-oriented medical chatbot designed to provide general health information and medical guidance. Powered by the Llama 3 8B large language model, it can understand natural language queries and generate helpful responses to common healthcare-related questions.

This system was developed as part of a thesis titled "Design and Implementation of a Task-Oriented Medical Chatbot using Llama 3 8B" by Baiyifan at Wenzhou Kean University.

Key Features

Symptom Information

Learn about common symptoms and their possible causes

Medication Guidance

General information about medications and safe usage

Healthcare Navigation

Guidance on when to see a doctor and what to expect

Natural Language Understanding

Powered by Llama 3 8B for intelligent conversations

Conversation History

All conversations are stored for continuity

Safety First

Emergency detection and medical disclaimers

Technology Stack

Python 3.11 Flask Llama 3 8B MySQL SQLAlchemy HTML5/CSS3 JavaScript REST API

System Architecture

The chatbot follows a task-oriented dialogue design with several key components:

  • Natural Language Processing Module
  • Llama 3 8B Language Model Integration
  • Task-Oriented Dialogue Management
  • MySQL Database for Conversation Storage
  • Responsive Web Interface

Research Background

This project explores how large language models can be integrated into task-oriented medical chatbot systems to improve healthcare accessibility. The study focuses on:

  • Designing an effective task-oriented dialogue structure for medical conversations
  • Integrating Llama 3 8B for natural language understanding and generation
  • Implementing prompt engineering to ensure safe and accurate responses
  • Building a complete web-based chatbot system with database integration

The system demonstrates the potential of LLMs in healthcare applications while maintaining safety through proper disclaimers and emergency detection.
