# Medical-Chatbot
Medical-Chatbot is a task-oriented medical chatbot based on the Llama 3 8B large language model. It helps users with symptom inquiry, medication information, emergency detection, and general health advice. The system is built with Python and Flask, using Ollama for local LLM deployment and MySQL for conversation storage. To run the project, first install Python 3.11 and Ollama, then pull the required models using ollama pull gemma3:4b. Clone the repository, create a virtual environment, install dependencies with pip install -r requirements.txt, and run python app.py. The chatbot achieves 85% response accuracy, 88% response clarity, 83% user satisfaction, 2.3 seconds average response time, and 96% emergency detection accuracy. Access the web interface at http://127.0.0.1:5000.
